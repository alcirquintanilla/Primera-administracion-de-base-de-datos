# 📚 Biblioteca - Primera Administración de Base de Datos I

## 📖 Descripción

Este proyecto corresponde a la primera administración de base de datos.

Consiste en el diseño e implementación de una base de datos para la gestión de una biblioteca, aplicando conceptos fundamentales del modelado relacional, creación de tablas, relaciones entre entidades y manipulación de datos mediante SQL.

---

## 🛠️ Tecnologías utilizadas

- SQL
- MySQL
- XAMPP
- phpMyAdmin

---

## 🗂️ Estructura de la Base de Datos

La base de datos está compuesta por las siguientes tablas:

- 👨‍💼 **Empleados**
  - Almacena la información del personal de la biblioteca.

- 👥 **Usuarios**
  - Registra los usuarios que pueden solicitar préstamos de libros.

- ✍️ **Autores**
  - Contiene la información de los autores de cada libro.

- 📚 **Libros**
  - Almacena los datos de los libros disponibles en la biblioteca.

- 🔄 **Préstamos**
  - Registra los préstamos realizados por los usuarios, relacionando libros y empleados.

---

## 🎯 Objetivos del proyecto

- Diseñar una base de datos relacional.
- Aplicar claves primarias y foráneas.
- Establecer relaciones entre tablas.
- Crear tablas utilizando SQL.
- Insertar registros de prueba.
- Practicar consultas básicas sobre la base de datos.
